# qus0in dev blog

개인 기술 블로그 프로젝트입니다.

Astro를 기반으로 콘텐츠를 관리하고, React와 shadcn/ui, Tailwind CSS를 조합해 블로그 화면을 구성합니다. 배포는 Cloudflare Workers Assets 기준으로 맞춰져 있습니다.

## Stack

- Astro
- React
- Tailwind CSS
- shadcn/ui
- Cloudflare Workers

## Commands

| Command | Description |
| :-- | :-- |
| `pnpm dev` | 개발 서버 실행 |
| `pnpm build` | 프로덕션 빌드 |
| `pnpm preview` | 빌드 결과 확인 |
| `pnpm deploy` | Cloudflare 배포 |

## Open Source

이 프로젝트는 다음 오픈소스를 사용합니다.

- Astro
- React
- Tailwind CSS
- shadcn/ui
- Radix UI
- Lucide
- Wrangler

각 오픈소스의 라이선스는 해당 프로젝트 저장소와 배포 패키지의 라이선스 표기를 따릅니다.
