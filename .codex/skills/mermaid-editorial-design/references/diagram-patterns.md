# Diagram Patterns

Use these composition patterns before inventing a custom Mermaid layout.

## Good Fits

- request or data flow: top-to-bottom sequence
- before and after comparison: two separate diagrams
- layered architecture: grouped vertical stacks
- lifecycle: short linear stages

## Mobile Safety Rules

- prefer `flowchart TB`
- keep labels short
- split wide comparisons into multiple diagrams
- increase spacing before reducing font size

## Visual Direction

- one accent color for the main path
- muted support nodes
- soft borders
- generous padding
